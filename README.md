# Mneme-open

Open (Apache-2.0) client SDKs, examples, API reference, and prebuilt
binaries for [Mneme](https://github.com/axisrobo/mneme). This repository is a
published mirror; the Mneme engine source is not included here.

- `clients/` - thin network clients (Python, TypeScript, Go, CLI)
- `examples/` - usage examples
- `docs/` - client-facing API reference
- `contracts/` - protocol schemas (JSON-RPC, MCP, REST/OpenAPI, gRPC proto)
- Prebuilt binaries are attached to GitHub releases (see `BINARY-LICENSE.md`).
